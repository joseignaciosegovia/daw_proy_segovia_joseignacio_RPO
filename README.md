Descripción del proyecto: El proyecto consiste en una aplicación web que gestiona las reservas online de las
pistas del polideportivo y del campo de fútbol en Moral de Calatrava.

Tecnologías usadas:
- Lenguajes de Programación. JavaScript para el lado del cliente y PHP para el lado del servidor.
- Sistemas de Gestión de Bases de Datos. MongoDB.
- Entorno de Desarrollo. Visual Studio Code.
- Despliegue. La aplicación se desplegará con Docker.
- Frameworks. Angular.
